# **App Name**: Lata Lalwani Music

## Core Features:

- Hero Section: Display artist's name, title, and background image with details provided
- Animated Background: Implement gradient background with floating musical notes animation
- Music Display: Showcase music with stylish text links, and navigate users to YouTube
- Contact Information: Provide links to artist's social media accounts, like Youtube, Instagram
- Content Sections: Structure website content with sections such as About, Journey, Achievements and Performance.

## Style Guidelines:

- Primary color: Deep maroon (#c2185b) for sophistication and musical passion.
- Background color: Dark gradient background from deep purple-blue to black (#121212).
- Accent color: Soft pink (#F48FB1) for highlighting elements and creating a gentle contrast.
- Body and headline font: 'Alegreya', a serif font to add an elegant and intellectual feel.
- Clean and center-aligned sections to enhance readability.
- Floating musical notes (♪ ♩ ♫ ♬) that rise upward in the background.
